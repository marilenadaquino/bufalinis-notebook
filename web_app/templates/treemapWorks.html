Treemap

In the chart below each block corresponds to an author cited in the notebook. The color and size of the blocks represent in a proportional way the frequency with which each author is cited. By hovering on each block corresponding to an author, the number of excerpts from works by the author is displayed. By clicking on the same block, a series of sub-blocks are displayed, each of which corresponds to a work cited in the notebook. Also in this case the colors and dimensions of the blocks represent in a proportional way the frequency with which each work is cited. By hovering on each of them, you get the number of excerpts from that specific work cited in the notebook.
